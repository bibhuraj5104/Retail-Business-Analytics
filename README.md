# 🛍️ Retail Business Analytics Dashboard

An end-to-end Retail Business Analytics project built using **PostgreSQL, SQL, Python, and Power BI**. This project analyzes customer shopping behavior, visualizes business performance, and automatically generates actionable business recommendations.

---

# 📌 Project Objective

The objective of this project is to help retail businesses make data-driven decisions by:

- Analyzing customer purchasing behavior
- Identifying revenue-generating categories
- Understanding customer demographics
- Tracking seasonal sales trends
- Generating AI-inspired business recommendations

---

# 📊 Dashboard Pages

## 1️⃣ Executive Dashboard

Displays high-level business KPIs including:

- Average Purchase Amount
- Average Review Rating
- Number of Customers
- Revenue by Category
- Revenue by Season
- Revenue by Top Locations
- Payment Method Distribution
- Subscription Status Analysis

---

## 2️⃣ Customer Analytics

Provides customer-level insights such as:

- Age Group Analysis
- Gender Analysis
- Shopping Frequency
- Category Preferences
- Seasonal Trends
- Payment Preferences

---

## 3️⃣ Business Recommendation Engine

Python automatically generates business recommendations based on the dataset.

Examples:

- Increase inventory for high-performing categories.
- Launch promotional campaigns during peak seasons.
- Improve logistics for preferred shipping methods.
- Target high-value customer segments.
- Increase customer loyalty initiatives.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/bibhuraj5104/Retail-Business-Analytics.git
```

### Install Python Libraries

```bash
pip install pandas numpy psycopg2 sqlalchemy
```

### Run Python Script

```bash
python notebook/insight_generator.py
```

### Open Dashboard

Open:

```
powerbi/customer_behavior_dashboard.pbix
```

Refresh the data to view the latest recommendations.
